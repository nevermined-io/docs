## Welcome to Nevermined

Nevermined Payments is a TypeScript SDK that enables AI builders to monetize their agents through flexible payment models. With support for both cryptocurrency and fiat payments, you can start accepting payments for your AI services in minutes.

## Installation

Install the Nevermined Payments SDK using your preferred package manager:

```bash npm
npm install @nevermined-io/payments
```

```bash yarn
yarn add @nevermined-io/payments
```

```bash pnpm
pnpm add @nevermined-io/payments
```

## Initialize the SDK

First, you'll need an API key from the [Nevermined App](https://app.nevermined.io). Once you have it, initialize the Payments client:

```typescript
import { Payments } from '@nevermined-io/payments'

const payments = Payments.getInstance({
  nvmApiKey: process.env.NVM_API_KEY,
  environment: 'production' // or 'testing' for development
})
```

## Create Your First Agent

Let's create a simple AI agent with a payment plan:

```typescript
import { 
  Payments, 
  getERC20PriceConfig, 
  getFixedCreditsConfig 
} from '@nevermined-io/payments'

async function createMyFirstAgent() {
  // Initialize payments
  const payments = Payments.getInstance({
    nvmApiKey: process.env.NVM_API_KEY,
    environment: 'production'
  })

  // Define agent metadata
  const agentMetadata = {
    name: 'My AI Assistant',
    tags: ['ai', 'assistant'],
    dateCreated: new Date(),
    description: 'A helpful AI assistant for various tasks'
  }

  // Define API endpoints
  const agentApi = {
    endpoints: [
      { POST: 'https://api.myservice.com/query' }
    ]
  }

  // Configure pricing (10 USDC)
  const priceConfig = getERC20PriceConfig(
    10_000_000n, // 10 USDC (6 decimals)
    '0xA0b86a33E6441c41F4F2B8Bf4F2B0f1B0F1C1C1C', // USDC address
    process.env.BUILDER_ADDRESS // Your wallet address
  )

  // Configure credits (100 queries)
  const creditsConfig = getFixedCreditsConfig(
    100n, // Total credits
    1n    // Credits per request
  )

  // Register agent and plan
  const { agentId, planId } = await payments.registerAgentAndPlan(
    agentMetadata,
    agentApi,
    priceConfig,
    creditsConfig
  )

  console.log(`Agent created: ${agentId}`)
  console.log(`Plan created: ${planId}`)
}
```

## Accept Payments

Integrate payment validation into your agent endpoint:

```typescript
import express from 'express'

const app = express()

app.post('/api/query', async (req, res) => {
  // Validate access
  const isValid = await payments.isValidRequest(
    req.body.planId,
    req.body.agentId,
    req.body.subscriberAddress,
    req.headers['x-nvm-query-signature']
  )

  if (!isValid) {
    // Return payment required response
    const paymentCard = await payments.getAgentPaymentCard(req.body.agentId)
    return res.status(402).json(paymentCard)
  }

  // Process the AI request
  const result = await processAIQuery(req.body.prompt)

  // Redeem credits
  await payments.redeemCredits(
    req.body.planId, 
    1n, // Credits to redeem
    generateProof(req, result)
  )

  res.json(result)
})
```

## Query an Agent

Here's how subscribers can purchase a plan and query your agent:

```typescript
async function queryAgent() {
  // Get agent details
  const agent = await payments.getAgent(agentId)
  
  // Purchase a plan
  const plan = agent.plans[0]
  await payments.orderPlan(plan.planId)
  
  // Get access credentials
  const credentials = await payments.getAgentHTTPOptions(plan.planId, agentId)
  
  // Query the agent
  const result = await payments.query(agentId, credentials, {
    prompt: "Hello, how can you help me?"
  })
  
  console.log(result)
}
```
