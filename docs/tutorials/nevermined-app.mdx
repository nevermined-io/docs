# How to use the Nevermined App?

The [Nevermined App](https://nevermined.app) is a web application that allows users to interact with the Nevermined protocol. The App provides a user-friendly interface to interact with the protocol, including the ability to register and discover AI agents, create and manage payment plans, and query AI agents.

You can find a complete guide on how to use the Nevermined App in the [Nevermined App documentation](https://docs.nevermined.app/docs/tutorials).
